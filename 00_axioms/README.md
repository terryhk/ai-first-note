# 开篇：基础公理与Aha时刻

> 不是术语表，是两个真实故事引出的底层公理——全书一切推演的根基

---

这本书不从一个术语表开始。

大多数AI入门书的第一页，都在堆名词——Transformer、Token、RAG、Agent——你还没进入正题，就先被一堵术语墙挡在门外。我不想那样。

我想从两个真实的故事开始。不是因为故事好听，而是因为这两个故事各自指向一条底层公理——全书此后的一切推演、一切方法、一切实战，都建在这两条公理之上。就像《经济学的思维方式》从一个路口的堵车场景引出"所有人在特定激励下做出选择"一样，我希望你读完这两个故事后，不用背任何定义，就能直觉地理解：AI时代，什么变了，什么没变，以及——你的位置在哪里。

## 本节内容

> **💡 阅读提示**：如果你已经有使用AI的丰富经验，可以快速浏览两个故事，重点看"从Aha时刻到基础公理"部分。

- **Steven的两周**：CFA转程序员，屎山项目月余无解，Claude Code两周搞定——稀缺性转移了
- **十五年的笔记**：投资笔记普通AI变糊，顶级模型体系化提炼出秩序——你的产权才是支点
- **从Aha时刻到基础公理**：稀缺性与极限替代公理 + 产权与资产协同公理
