# 本章小结

## 核心结论

Token和Embedding是AI世界的"经济学基础设施"——Token是你的预算约束，Embedding是语义地图。它们共同定义了这个世界的稀缺性、成本和搜索方式。

## 本章思维习惯

**预算与坐标思维**——每次使用AI前，问自己三个问题：

1. **预算问题**：这个任务值得花多少Token？边际收益是否超过边际成本？
2. **坐标问题**：AI真的"理解"我的问题吗？还是只是在语义空间中找了最接近的点？
3. **效率问题**：是否可以通过精选上下文、分步骤执行、选对模型来优化Token投入回报率？

## 检查清单

- [ ] 是否把最关键的信息放到了提示词的开头和结尾？
- [ ] 是否评估了这个任务的边际收益是否超过边际成本？
- [ ] 是否选择了合适价位的模型（简单任务用便宜模型，复杂任务用顶级模型）？
- [ ] 是否理解了你使用的AI系统的上下文窗口大小？
- [ ] 是否意识到语义搜索找到的内容只是"统计上相近"而非"确定相关"？
