# 第2章 稀缺的硬通货：Token与语义坐标

> Token是AI世界的"钱"，Embedding是AI世界的"地图"——搞懂这两样，你就懂了AI的"经济学基础设施"

---

**本章心法：** 在AI的世界里，Token是预算约束（你有多少钱），Embedding是地理坐标（事物在哪里）。搞懂这两样，你就懂了AI的"经济学基础设施"。

如果你用过ChatGPT或Claude，你大概率遇到过这种情况：给AI塞了一整份文档，它却漏掉了最关键的部分。或者听说"Token是计费的"，但搞不懂为什么同样的字数在不同场景下费用差那么多。

这一章拆解AI世界的两个基础概念——Token和Embedding。它们不是需要你记住的技术术语，而是理解AI行为模式的"经济学基础设施"。搞懂了Token，你就知道为什么"少即是多"；搞懂了Embedding，你就知道AI是怎么"理解"你的问题的。
