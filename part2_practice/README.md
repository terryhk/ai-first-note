# 第二部分：AI大模型的实践笔记

> 核心哲学：**半人马协作——人类做判断，AI做执行。**
> 每完成一次实践 = 一次概率迭代。
> 技术武器逐章升级：基础Prompt → 思维链 → RAG → ReAct Agent → PAL → 半人马决策。
> 每章一个真实场景，看完一章，明天就能用上。

---

如果你从第一部分一路读到这里，你已经建立了完整的判断框架。

你知道AI的本质是概率采样系统——它像幽灵不像狗，你会驾驭它而不是驯服它。你懂Token的经济学——上下文是预算，你要把注意力花在回报最高的位置。你学会了写Prompt合同——激励约束、4S原则、思维链、三层架构。你了解了工具生态——从Prompt到微调，不同的P和成本组合。你掌握了贝叶斯裁判法——不凭直觉，靠证据。

这些知识很强大，但有一个问题：**你还没有真正用过。**

第二部分就是来填这个缺口的。它不是再教你概念，而是带你做任务——每章一个真实的、可复用的AI实践场景。

这一部分的结构以你的真实需求为中心，而非以技术分类为中心：

- **第6章 个人知识库搭建**：把散落的笔记变成你的第二大脑，让笔记自己来找你
- **第7章 AI辅助学习**：费曼学习法×AI，温故而知新的迭代秘密
- **第8章 AI赋能产品工作**：周报、规范、分析与日常提效
- **第9章 文档处理与产出**：从阅读到产出，30页报告一上午搞定
- **第10章 数据分析**：不写代码也能洞察，PAL方法入门
- **第11章 Agent自动化**：从ReAct到放手
- **第12章 深度研究**：多源搜索与偏见检测
- **第13章 辅助决策**：从纠结到清晰

每一章都配有概率工具箱、边际成本与机会成本分析、可复用的Prompt模板。你不需要在读完第一章后就记住所有东西——用到哪章翻到哪章，把这本书当作你的AI实践手册。

**记住：你每次完成一次实践，就是在做一次概率迭代。你写Prompt/搭流程（行为）→ 跑任务（概率）→ 评估修正 → 再跑。循环往复，你的成功率一步步逼近100%。**
