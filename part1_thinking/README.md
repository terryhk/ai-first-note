# 第一部分：AI大模型的思维方式

> 理解AI的概率本质，建立判断框架

---

第一部分解决一个问题：**AI到底是什么？**

不是"它有多厉害"——那是营销话术。而是：它的本质机制是什么？它的能力边界在哪里？它为什么会出错？你怎么控制它？

这一部分的逻辑线，对应经济学的层层递进：

- **谜题**（第1章）：为什么AI不完美？→ 因为它本质是概率机器
- **稀缺约束**（第2章）：AI理解世界的最小单位是什么？→ Token与语义坐标
- **契约**（第3章）：怎么让AI按你的意思办事？→ Prompt就是合同
- **市场**（第4章）：AI的工具生态怎么用？→ 从赤手空拳到工具外挂
- **宏观调控**（第5章）：怎么评估AI做得好不好？→ 贝叶斯裁判法

读完第一部分，你会建立一套判断框架——面对任何AI输出，你知道它在概率分布的什么位置，该信多少，该怎么改进。这是全书的根基。
