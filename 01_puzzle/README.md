# 第1章 谜题：为什么世界上没有完美的AI？

> AI不完美不是bug，是概率的本质——理解了这一点，你的AI焦虑会消失一半

---

**本章心法：** 行为 × 概率 = 结果。你控制行为（Prompt/流程），AI控制概率（采样/幻觉）。正确的策略不是一次写对，是持续提升成功的概率。

如果你用过AI，你大概率经历过两种困惑：同一个问题问了三次，得到三个完全不同但每个看起来都"有道理"的答案；AI用极其自信的语气告诉你一个完全错误的事实。这两个现象不是bug，它们是AI的本质特征——就像水往低处流不是河流的bug，是重力的必然结果。
